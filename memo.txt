spring websocket을 활용한 1대1 채팅

프로토콜 정의
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
방향			동작		명령		파라미터				처리
==============================================================================================================================================================================
클라이언트 -> 서버	대화요청하기	CONNECT		상대방아이디				대화를 나눌 상대방 아이디를 서버로 전송, 연결요청 버튼 비활성화
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
클라이언트 -> 서버	대화종료하기	CLOSE		상대방아이디				대화를 종료할 상대방 아이디를 서버로 전송
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
클라이언트 -> 서버	메세지송신하기	MSG		상대방아이디:메세지			입력 메세지를 서버로 전송
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
서버 -> 클라이언트	접속자정보	ON_USERS	아이디,아이디,아이디			사용자 목록에서 접속자를 찾아서 온라인으로 표시, 연결요청 버튼 활성화
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
서버 -> 클라이언트	접속종료자정보	OFF_USER	아이디					사용자 목록에서 해당 사용자를 오프라인으로 표시, 연결요청 버튼 비활성화
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
서버 -> 클라이언트	대화요청자정보	ON_CHAT		대화요청자아이디			대화상자를 화면에 표시하고, 연결요청 버튼 비활성화
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
서버 -> 클라이언트	대화종료자정보	OFF_CHAT	대화종료자아이디			대화상자를 화면에서 제거하고, 연결요청 버튼 활성화
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
서버 -> 클라이언트	대화정보	CHAT_MSG	상대방아이디:보낸사람아이디:메세지	대화상자에 메세지 표시
==============================================================================================================================================================================
